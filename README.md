# DownloaderPRO
DownloaderPRO: GUI moderna em HTA/JS para download de vídeos e playlists via yt-dlp. Utiliza ActiveX/WScript.Shell para orquestração de scripts Batch/PowerShell e Winget para gestão de dependências nativas no Windows. Arquitetura portátil com caminhos relativos (%~dp0), garantindo execução segura e transparente sem binários ocultos.
